# Portfolio Website - Botu Akhil Sai

This is a single-file HTML portfolio website for Botu Akhil Sai.

## How to Run

1.  Simply double-click the `index.html` file to open it in your web browser.
2.  That's it! No installation or extra software is required.

## Features

-   **Hero Section**: Galaxy theme with parallax effects.
-   **Bento Grid**: Quick info about Akhil.
-   **Automation Architect**: AI-powered solution generator using Gemini API.
-   **Content**: Experience, Tech Stack, Projects, Education.

## Technologies Used

-   **HTML5**
-   **Tailwind CSS** (via CDN)
-   **Lucide Icons** (via CDN)
-   **Vanilla JavaScript**
